INSTRUCTIONS
=============
Using the Booking site:
1. Open booking.html
2. Input information and click send or press enter
3. Click admin to navigate to the admin site (this is for convenience of navigation only)

Using the Admin site:
1. Open admin.html or navigate from booking.html
2. Click the search button to display all pick-up requests with unassigned requests within 2 hours from now
3. Input the reference number and click assign to assign a taxi to the booking number
